# 📚 Vinny Web — موقع الروايات

موقع روايات عربي متكامل بتصميم داكن أنيق.

## 🚀 كيفية النشر على GitHub Pages (من الهاتف)

### الخطوات:

1. **أنشئ حساب GitHub** على [github.com](https://github.com) إذا لم يكن لديك

2. **أنشئ مستودع جديد:**
   - اضغط على **+** ثم **New repository**
   - سمّه: `vinny-web`
   - اختر **Public**
   - اضغط **Create repository**

3. **ارفع الملفات:**
   - في صفحة المستودع اضغط **Add file → Upload files**
   - ارفع جميع الملفات بنفس البنية:
     ```
     index.html
     novels.html
     novel.html
     chapter.html
     categories.html
     admin.html
     css/style.css
     js/data.js
     js/app.js
     ```
   - اضغط **Commit changes**

4. **فعّل GitHub Pages:**
   - اذهب إلى **Settings** في المستودع
   - من القائمة الجانبية اختر **Pages**
   - في **Source** اختر **Deploy from a branch**
   - اختر **main** ثم **/ (root)**
   - اضغط **Save**

5. **موقعك جاهز!** بعد دقيقتين يكون على:
   `https://اسم_حسابك.github.io/vinny-web`

## ✨ المميزات

### للزوار:
- 🏠 صفحة رئيسية مع روايات مميزة وأحدث الإضافات
- 📚 مكتبة مع بحث وفلترة متقدمة
- 📖 صفحة تفاصيل رواية مع قائمة الفصول
- 📄 صفحة قراءة فصل مع:
  - شريط تقدم القراءة
  - تكبير/تصغير الخط
  - وضع القراءة الليلي/النهاري
  - تنقل بين الفصول
- 🏷️ تصنيفات تفاعلية

### للإدارة (`/admin.html`):
- ➕ إضافة/تعديل/حذف روايات
- 📝 إضافة/تعديل فصول
- 🏷️ إدارة التصنيفات
- 📊 إحصائيات مباشرة

## 💾 قاعدة البيانات

يستخدم الموقع **LocalStorage** للتخزين — لا تحتاج إلى سيرفر!
كل بيانات الروايات والفصول تُحفظ في متصفح المستخدم.

> **ملاحظة:** لأن البيانات تُحفظ في المتصفح، كل جهاز له بياناته الخاصة.
> إذا أردت مشاركة المحتوى مع الزوار، استبدل LocalStorage بـ Firebase أو Supabase لاحقاً.
